# Archived Unwanted Files

This folder contains files that are not part of the active deployment path anymore.

## What lives here

- Old lint output reports
- Reference exports that duplicate `real_data.md`
- Unsafe or redundant development scripts
- Unused public/template assets that are not referenced by the current app

## Why they were moved

- They created root/public clutter without helping the live product.
- Some of them contained temporary or unsafe development-only content.
- Keeping them here preserves history while making the deployable project cleaner.
